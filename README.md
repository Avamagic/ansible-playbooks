# Ansible playbooks

This repository contains Ansible playbooks used by [Avamagic][].

Playbooks are organized by directories. Please refer to the README
within each directories for their descriptions.

## Ansible

[Ansible][] is a model-driven configuration management, multi-node
deployment/orchestration, and remote task execution system. It is
required on your local machine to execute the playbooks. Please refer
to [Getting Started][] on installing Ansible.

[Ansible]: http://ansible.cc/
[Avamagic]: http://avamagic.com/
[Getting Started]: http://ansible.cc/docs/gettingstarted.html
